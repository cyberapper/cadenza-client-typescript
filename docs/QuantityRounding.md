# QuantityRounding

Quantity rounding strategy to meet lot size requirements. Empty string means no rounding (rejects orders that don\'t meet lot size requirements).

## Enum

* `Empty` (value: `''`)

* `Up` (value: `'UP'`)

* `Down` (value: `'DOWN'`)

* `Nearest` (value: `'NEAREST'`)

* `Ceil` (value: `'CEIL'`)

* `Floor` (value: `'FLOOR'`)

* `HalfUp` (value: `'HALF_UP'`)

* `HalfDown` (value: `'HALF_DOWN'`)

* `HalfEven` (value: `'HALF_EVEN'`)
